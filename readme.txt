Данный модуль интеграции с kaznachey разработан для 1C-BITRIX.
 
Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "ПОДКЛЮЧЕНИЕ"). 
После подписания договора будут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 
Переданные данные будут использованы при настройки модуля

Порядок установки модуля:
1. Распакуйте архив в любое удобное Вам место;
2. Скопируйте модуль в каталог …\www\bitrix\modules\sale\payment;
3. Разместить файл kaznachey_result.php из папки модуля kaznachey в любом месте сайта (доступного из вне) и указать на него ссылку в Result URL в панели управления kaznachey, например можете разместить файл в корне сайта.
4. Зайдите в администрирование BITRIX;
5. Выберите закладку «Панель управления»;
6. Выберите пункт «Магазин»;
7. Раскройте ветку дерева «Настройки магазина»;
8. Выберите пункт «Платежные системы»;
9. Нажмите кнопку «Добавить платежную систему»;
10. В качестве валюты укажите «UAH», выберите галочку «Активность»;
11. В поле «Название» впишите «kaznachey»;
12. Нажмите кнопку «Применить»;
13. Перейдите на закладку «Физическое лицо»;
14. Установите флажок «Применять для данного типа плательщика»;
15. В поле «Название» впишите «kaznachey»;
16. В поле «Обработчик» из списка выберите «kaznachey»;
17. Заполните все поля значениями из настроек панели управления магазином на сайте kaznachey (не забывайте корректно устанавливать тип параметров);
18. Нажмите кнопку «Применить»;
19. Перейдите на закладку «Юридическое лицо»;
20. Повторите все действия, как с закладкой «Физическое лицо» (п.13-17);
21. Перейдите на закладку «Платежная система» и установите флажок напротив поля «Активность»;
22. Установите тестовый режим работы.
23. Нажмите кнопку «Сохранить».
